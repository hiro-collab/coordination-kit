# Maintainers

Use this folder only when you are maintaining or publishing the kit itself.

Read this folder in order:

1. [publishing-checklist.md](publishing-checklist.md)
2. [../CHANGELOG.md](../CHANGELOG.md)

## What Is Here

- `publishing-checklist.md`: release readiness and GitHub publishing checks
- `../CHANGELOG.md`: notable changes to the kit

## Rule

Keep kit maintenance separate from live trial execution unless a safety issue forces an immediate fix.
