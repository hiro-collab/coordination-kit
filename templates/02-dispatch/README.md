# Dispatch Templates

Use these files when someone must read a file and take action, not just store information.

Read this folder in order:

1. `COMMAND_DISPATCH_TEMPLATE.md`
2. `SEND_READY_PROMPT_TEMPLATE.md`
3. `FINGER_TASK_CARD_TEMPLATE.md`

## What Each File Is For

- `COMMAND_DISPATCH_TEMPLATE.md`: explicit read-and-act instruction to a responsible thread
- `SEND_READY_PROMPT_TEMPLATE.md`: exact prompt body for a human forwarder to paste
- `FINGER_TASK_CARD_TEMPLATE.md`: scoped assignment for a finger or lower node

## Standard Finger Names

When five finger lanes are active, name them `Thumb`, `Indy`, `Middy`, `Ringy`, and `Pinky`.

Use compact node IDs such as `A.Thumb`, `A.Indy`, and `B.Shoulder` when reporting or assigning work.
