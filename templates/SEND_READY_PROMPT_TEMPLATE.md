# Send-Ready Prompt Template

Use this when a human operator must copy and forward a message to another thread.

- Owner:
- Readers:

```text
Recipient:

Read First:
- <absolute or project-relative file path>

Purpose:
- <why this is being sent>

Do Now:
- <what to do immediately>

Update Or Submit To:
- <destination file or thread-visible output>

Required Section Or Format:
- <exact section or reply format>

Do Not Do:
- <scope guard>

Reply:
[<node>] status
Task:
Done:
Blocked:
Next:
```

Notes:

- If you ask a user or operator to forward a message, provide the full send-ready body inline.
- A file path alone is reference material, not a sufficient forwarding instruction.
