# Commander Sign-Off Template

Use this at the end of a run.

- Owner:
- Readers:

```text
[Commander] sign-off
Mission:
- <mission summary>

Acceptance:
- <criterion>: <passed|failed|waived>

Accepted Deviations:
- <deviation>

Residual Gaps:
- <gap>

Rejected Or Deferred Items:
- <item>

Decision:
- <accepted|rework required>

Reason:
- <reason>

Separate Judgments If Useful:
- Preferred output:
- Operations winner:
- Overall result:
```

Optional competitive-trial add-on:

- Replace `Preferred output` with `Game winner` when the run is a comparative game trial.
