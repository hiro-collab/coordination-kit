# Handoff Templates

Use these files near the end of a run when participants must submit, review, and sign off with visible evidence.

Skip the profile-specific scorecard when the run is not a comparative game trial.

Read this folder in order:

1. `SUBMISSION_PACKET_TEMPLATE.md`
2. `FINAL_HANDOFF_TEMPLATE.md`
3. `COMMANDER_SIGNOFF_TEMPLATE.md`
4. `GAME_JUDGING_SCORECARD_TEMPLATE.md`
5. `KIT_RETROSPECTIVE_TEMPLATE.md`

## What Each File Is For

- `SUBMISSION_PACKET_TEMPLATE.md`: local submission draft before the final shared handoff update
- `FINAL_HANDOFF_TEMPLATE.md`: explicit final handoff for launch method, cuts, and known defects
- `COMMANDER_SIGNOFF_TEMPLATE.md`: Commander acceptance or rejection of the final state
- `GAME_JUDGING_SCORECARD_TEMPLATE.md`: optional evidence-based judging form for comparative game trials
- `KIT_RETROSPECTIVE_TEMPLATE.md`: post-run improvement capture for the kit itself
